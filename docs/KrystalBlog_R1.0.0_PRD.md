## 一、项目概述

### 1.1 项目定位

KrystalBlog 是一个面向个人创作者的综合型个人博客 Web 平台，涵盖文章、视频、音乐、文件存储等多种内容形式，支持访客浏览互动与管理员内容管理，旨在为创作者提供一个统一、美观、易用的个人内容展示与管理中心。

### 1.2 项目目标

- 为管理员（内容创作者）提供高效的多类型内容管理能力
- 为普通访客提供流畅、美观的内容浏览与互动体验
- 提供完整的数据统计能力，帮助创作者了解内容传播效果

### 1.3 核心价值

| 对象 | 核心价值 |
|------|----------|
| 管理员 | 一站式管理博客文章、视频、音频、文件，操作简单高效 |
| 普通用户 | 优质内容浏览体验，支持搜索、评论、点赞等互动 |

## 三、用户角色定义

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 管理员（Admin） | 网站所有者 / 创作者，唯一拥有内容管理权限的账号 | 全部功能，包含内容增删改、网盘、数据统计 |
| 普通用户（Guest/User） | 访问网站的普通访客，无需登录可浏览，登录后可互动 | 浏览、搜索、评论、点赞、播放 |

> **说明：** 本期评论和点赞功能需判断用户登录状态，未登录用户引导注册/登录后操作。

## 四、UI 设计规范

### 4.1 整体风格

- **风格关键词：** 简约、温暖、大气、现代
- **主色调：** 暖色系为主（如暖米白、浅橙、深棕金等），搭配中性灰辅助色
- **字体：** 中文使用无衬线体（如思源黑体），英文使用现代无衬线体

### 4.2 交互动效

- 页面切换使用淡入淡出或滑动过渡，时长 200–400ms
- 卡片悬浮时有轻微阴影上浮动效
- 按钮点击有微缩反馈动画
- 媒体加载时使用骨架屏（Skeleton），避免白屏

### 4.3 布局原则

- 响应式设计，适配桌面端和移动端
- 信息层级清晰，重要内容突出展示
- 操作路径不超过 3 步，核心功能入口常驻导航栏

## 五、功能需求 — 博客文章模块

### 5.1 功能概览

| 功能 | 管理员 | 普通用户 |
|------|--------|----------|
| 新增文章 | ✅ | ❌ |
| 编辑文章 | ✅ | ❌ |
| 删除文章 | ✅ | ❌ |
| 文章列表管理 | ✅ | ❌ |
| 查看文章详情 | ✅ | ✅ |
| 搜索文章 | ✅ | ✅ |
| 按分类/标签/归档浏览 | ✅ | ✅ |
| 评论文章 | ✅ | ✅（需登录） |
| 点赞文章 | ✅ | ✅（需登录） |

### 5.2 管理员功能

#### 5.2.1 新增文章

- 进入管理后台 → 文章管理 → 新建文章
- 编辑器支持 **Markdown** 格式，提供实时预览
- 必填字段：标题、正文内容、分类
- 可选字段：标签（多选）、封面图、摘要、是否公开发布、发布时间（支持定时发布）
- 保存时校验：标题不为空、正文不为空
- 支持保存为草稿，草稿不对外展示

#### 5.2.2 编辑文章

- 在文章管理列表中点击"编辑"进入编辑页
- 编辑界面与新增界面一致，回填已有内容
- 保存后立即更新线上展示内容

#### 5.2.3 删除文章

- 支持单条删除，操作前弹出二次确认弹窗："确认删除该文章？删除后不可恢复"
- 支持批量删除，勾选多条后点击"批量删除"
- 删除后对应的评论、点赞数据一并清除

#### 5.2.4 文章列表管理

- 支持按标题关键词搜索
- 支持按分类、标签、发布状态（已发布/草稿）、时间范围筛选
- 列表展示字段：标题、分类、标签、阅读数、评论数、点赞数、发布时间、状态

### 5.3 普通用户功能

#### 5.3.1 文章列表

- 首页展示最新文章卡片列表，每页 10 条，支持分页
- 卡片展示：封面图（无封面展示默认图）、标题、摘要、分类、标签、发布日期、阅读数

#### 5.3.2 文章详情

- 渲染 Markdown 为 HTML 展示，支持代码高亮、数学公式（KaTeX/MathJax 可选）
- 展示：文章标题、作者、发布时间、阅读数、点赞数、正文内容、标签
- 右侧或底部展示目录（TOC），支持点击跳转
- 阅读时自动累加阅读数（每个 IP/Session 计一次）

#### 5.3.3 搜索

- 支持关键词全文搜索（标题、正文）
- 展示搜索结果列表，结果中关键词高亮显示
- 搜索无结果时展示友好提示

#### 5.3.4 分类 / 标签 / 归档

- **分类页：** 展示所有文章分类及对应文章数量，点击分类进入该分类文章列表
- **标签页：** 标签云形式展示所有标签，点击标签进入该标签文章列表
- **归档页：** 按年月时间轴展示文章，点击月份展开该月文章列表

#### 5.3.5 评论

- 文章详情页底部展示评论区
- 支持一级评论和楼层回复（最多两级）
- 评论需登录，未登录点击评论框提示"请先登录"
- 评论内容字数限制：1–500 字
- 评论提交后实时刷新评论列表
- 管理员可在后台删除任意评论

#### 5.3.6 点赞

- 文章详情页展示点赞按钮及点赞数
- 同一用户对同一文章只能点赞一次，再次点击取消点赞
- 未登录用户点击点赞提示"请先登录"

### 5.4 边界与异常

| 场景 | 处理方式 |
|------|----------|
| Markdown 渲染异常 | 展示原始文本，同时上报错误日志 |
| 文章不存在（已删除） | 返回 404 页面，提示"文章不存在或已删除" |
| 搜索结果为空 | 展示"暂无相关内容"提示，推荐热门文章 |
| 网络异常 | 展示通用错误提示，提供刷新按钮 |

## 六、功能需求 — 视频模块

### 6.1 功能概览

| 功能 | 管理员 | 普通用户 |
|------|--------|----------|
| 上传视频 | ✅ | ❌ |
| 删除视频 | ✅ | ❌ |
| 视频管理 | ✅ | ❌ |
| 查看/播放视频 | ✅ | ✅ |
| 搜索视频 | ✅ | ✅ |
| 按分类/标签浏览 | ✅ | ✅ |
| 评论视频 | ✅ | ✅（需登录） |
| 点赞视频 | ✅ | ✅（需登录） |

### 6.2 管理员功能

#### 6.2.1 上传视频

- 支持常见视频格式：MP4、AVI、MKV、MOV（后端转码统一为 MP4/H.264）
- 上传时填写：视频标题（必填）、分类（必填）、标签（可选）、简介（可选）、封面图（可选，不填则自动截取首帧）
- 上传显示进度条，支持取消上传
- 单个视频文件大小上限：**2GB**（可在配置中调整）

#### 6.2.2 删除视频

- 单条删除，操作前弹出二次确认
- 删除后同步清除对应评论、点赞、播放记录

#### 6.2.3 视频管理列表

- 展示字段：封面图、标题、分类、标签、播放数、评论数、点赞数、上传时间
- 支持按标题搜索、分类/标签筛选

### 6.3 普通用户功能

#### 6.3.1 视频列表

- 卡片展示视频封面、标题、分类、播放数、上传时间
- 支持分页浏览

#### 6.3.2 视频播放

- 内嵌 HTML5 播放器，支持播放/暂停、进度拖拽、音量调节、全屏
- 播放时自动累加播放数
- 视频加载时展示加载动画

#### 6.3.3 搜索、分类、标签

- 关键词搜索（标题、简介）
- 分类页、标签页与博客文章模块逻辑一致

#### 6.3.4 评论 / 点赞

- 逻辑与博客文章模块评论/点赞一致

### 6.4 边界与异常

| 场景 | 处理方式 |
|------|----------|
| 视频格式不支持 | 上传时前端校验，提示支持的格式 |
| 视频加载失败 | 展示错误提示，提供"重新加载"按钮 |
| 文件超过大小限制 | 上传前端校验，提示文件大小超限 |

## 七、功能需求 — 听歌模块

### 7.1 功能概览

| 功能 | 管理员 | 普通用户 |
|------|--------|----------|
| 上传音频 | ✅ | ❌ |
| 删除音频 | ✅ | ❌ |
| 音频管理 | ✅ | ❌ |
| 查看歌曲列表 | ✅ | ✅ |
| 搜索歌曲 | ✅ | ✅ |
| 播放音频 | ✅ | ✅ |
| 按分类/标签/作者浏览 | ✅ | ✅ |

### 7.2 管理员功能

#### 7.2.1 上传音频

- 支持格式：MP3、FLAC、WAV、AAC
- 上传时填写：歌曲名称（必填）、歌手/作者（必填）、分类（必填）、标签（可选）、封面图（可选）、歌词文件 LRC（可选）
- 单个文件大小上限：**200MB**

#### 7.2.2 删除音频

- 单条删除，操作前二次确认

#### 7.2.3 音频管理列表

- 展示字段：封面、歌曲名、歌手、分类、标签、播放数、上传时间

### 7.3 普通用户功能

#### 7.3.1 歌曲列表

- 卡片或列表形式展示，显示封面、歌曲名、歌手、时长

#### 7.3.2 播放器

- 页面底部常驻音频播放栏（类音乐 App 样式）
- 支持：播放/暂停、上一首/下一首、进度拖拽、音量调节、循环/随机模式切换
- 支持 LRC 歌词同步滚动展示（如有歌词文件）
- 切换页面时播放不中断

#### 7.3.3 搜索

- 支持按歌曲名称、歌手/作者关键词搜索

#### 7.3.4 按作者 / 分类 / 标签浏览

- **作者页：** 按歌手/作者分组展示歌曲
- **分类/标签：** 逻辑与博客文章模块一致

### 7.4 边界与异常

| 场景 | 处理方式 |
|------|----------|
| 音频格式不支持 | 前端上传校验，提示支持格式 |
| 无歌词文件 | 播放器歌词区显示"暂无歌词" |
| 播放器切换歌曲失败 | 展示错误提示，自动跳至下一首 |

## 八、功能需求 — 网盘模块

### 8.1 权限说明

网盘模块**仅管理员可见和使用**，普通用户无任何网盘相关入口。

### 8.2 功能需求

#### 8.2.1 文件上传

- 支持任意文件类型上传
- 支持多文件同时上传，显示上传进度
- 上传完成后文件出现在文件列表中

#### 8.2.2 文件下载

- 点击文件名或"下载"按钮触发下载
- 支持批量下载（打包为 ZIP）

#### 8.2.3 文件管理

- 支持创建文件夹，文件夹支持层级嵌套
- 支持文件重命名
- 支持单文件/批量删除，删除前二次确认
- 支持文件移动（拖拽或选择目标文件夹）
- 文件列表展示：文件名、文件类型图标、文件大小、上传时间

#### 8.2.4 搜索

- 支持按文件名关键词搜索

### 8.3 边界与异常

| 场景 | 处理方式 |
|------|----------|
| 存储空间不足 | 上传前校验，提示剩余空间不足 |
| 文件名重复 | 提示文件名已存在，询问是否覆盖或重命名 |
| 上传中断 | 显示上传失败提示，支持重试 |

## 九、功能需求 — 友链模块

### 9.1 功能概述

友链（Friend Links）模块用于展示与其他博主 / 网站的互链，增加站点对外曝光与社区连接感。

### 9.2 功能概览

| 功能 | 管理员 | 普通用户 |
|------|--------|----------|
| 添加友链 | ✅ | ❌ |
| 编辑友链 | ✅ | ❌ |
| 删除友链 | ✅ | ❌ |
| 审核友链申请 | ✅ | ❌ |
| 查看友链列表 | ✅ | ✅ |
| 申请友链 | ❌ | ✅ |

### 9.3 管理员功能

#### 9.3.1 添加友链

- 填写字段：站点名称（必填）、站点 URL（必填）、站点 Logo URL（可选）、站点描述（可选）、排序权重（可选）

#### 9.3.2 编辑 / 删除友链

- 支持修改所有字段
- 删除前二次确认

#### 9.3.3 审核友链申请

- 普通用户提交的友链申请以待审核状态存入后台
- 管理员可一键通过或拒绝，拒绝时可填写原因（该原因不对外展示，仅记录）
- 通过后自动出现在友链展示页

### 9.4 普通用户功能

#### 9.4.1 查看友链

- 友链页以卡片网格形式展示，每张卡片包含：站点 Logo、站点名称、站点描述
- 点击卡片以新标签页打开对应站点

#### 9.4.2 申请友链

- 页面底部提供"申请友链"入口
- 申请表单字段：申请方站点名称（必填）、站点 URL（必填）、站点 Logo URL（可选）、联系邮箱（必填）、备注（可选）
- 提交后展示"申请已提交，等待管理员审核"提示

### 9.5 边界与异常

| 场景 | 处理方式 |
|------|----------|
| 站点 URL 格式错误 | 前端正则校验，提示"请输入有效的 URL" |
| 重复申请同一 URL | 后端去重校验，提示"该站点已存在或已申请" |
| Logo 链接无法访问 | 展示默认 Logo 占位图 |

## 十、功能需求 — 数据统计模块

### 10.1 权限说明

数据统计模块**仅管理员可见**，入口在管理后台首页仪表盘。

### 10.2 博客文章数据统计

| 统计指标 | 说明 |
|----------|------|
| 总文章数 | 所有已发布文章数量 |

| 今日阅读量 | 当日文章总阅读数 |
| 阅读量 Top 10 | 阅读数最高的 10 篇文章排行 |
| 每日阅读趋势 | 近 30 天每天的阅读数折线图 |
| 分类文章分布 | 各分类下的文章数量占比饼图 |
| 标签使用频率 | 使用最多的 Top 20 标签 |
| 总评论数 | 全站文章累计评论数 |
| 总点赞数 | 全站文章累计点赞数 |

### 10.3 视频播放数据统计

| 统计指标 | 说明 |
|----------|------|
| 总视频数 | 所有已上传视频数量 |
| 总播放量 | 全站视频累计播放次数 |
| 今日播放量 | 当日视频总播放数 |
| 播放量 Top 10 | 播放数最高的 10 个视频排行 |
| 每日播放趋势 | 近 30 天每天的播放数折线图 |
| 分类视频分布 | 各分类下的视频数量占比 |
| 总评论数 | 全站视频累计评论数 |
| 总点赞数 | 全站视频累计点赞数 |

### 10.4 歌曲播放数据统计

| 统计指标 | 说明 |
|----------|------|
| 总歌曲数 | 所有已上传歌曲数量 |
| 总播放量 | 全站歌曲累计播放次数 |
| 今日播放量 | 当日歌曲总播放数 |
| 播放量 Top 10 | 播放数最高的 10 首歌曲排行 |
| 每日播放趋势 | 近 30 天每天的歌曲播放数折线图 |
| 歌手播放分布 | 各歌手/作者累计播放量对比柱状图 |
| 分类播放分布 | 各音乐分类的播放占比 |

### 10.5 网站基础信息统计

| 统计指标 | 说明 |
|----------|------|
| 今日访问量（PV） | 当日页面访问次数 |
| 今日独立访客（UV） | 当日独立 IP/Cookie 数量 |
| 累计访问量（PV） | 建站以来总访问次数 |
| 累计独立访客（UV） | 建站以来总独立访客数 |
| 每日访客趋势 | 近 30 天 PV/UV 双折线图 |
| 注册用户总数 | 平台注册账号总数 |
| 今日新增用户 | 当日新注册账号数 |
| 网站运行天数 | 从上线日期至今的天数 |
| 存储占用统计 | 视频、音频、网盘文件各自占用存储空间 |

### 10.6 仪表盘布局

- 顶部展示核心数字卡片（今日阅读、今日播放、今日访客等）
- 中部展示各模块趋势折线图
- 底部展示各模块 Top 10 排行榜

## 十一、非功能需求

### 11.1 性能要求

| 指标 | 要求 |
|------|------|
| 普通页面首屏加载 | ≤ 2 秒（正常网络环境） |
| API 接口响应时间 | 常规接口 ≤ 500ms，统计接口 ≤ 2s |
| 视频/音频起播延迟 | ≤ 3 秒 |
| 并发用户支持 | 支持 100 并发访问（个人博客基准） |

### 11.2 安全要求

- 管理员账号密码加密存储（bcrypt）
- API 接口鉴权使用 JWT Token
- 文件上传限制类型白名单（视频、音频模块），网盘模块信任管理员不限制
- 防止 XSS：富文本/Markdown 渲染前过滤危险标签
- 防止 SQL 注入：使用参数化查询，禁止拼接 SQL

### 11.3 兼容性

- 浏览器兼容：Chrome、Edge、Firefox、Safari 最新两个大版本
- 设备兼容：响应式支持 PC 端（≥1024px）、平板端（768px–1023px）、手机端（<768px）

### 11.4 可维护性

- 前端组件化开发，公共组件统一管理
- 后端接口遵循 RESTful 规范，提供 Swagger API 文档
- 关键操作（上传、删除、登录）记录操作日志
